# sitebuilder

Static GitHub Pages prototype for the golf website admin dashboard.

## Files

- `index.html`: GitHub Pages entrypoint.
- `golf_website_admin_mockup.jsx`: React prototype rendered in-browser.

## Publish to GitHub Pages

1. Push this folder to `https://github.com/twomuchjefferson/sitebuilder`.
2. In GitHub, open `Settings` > `Pages`.
3. Set `Source` to `Deploy from a branch`.
4. Select the `main` branch and `/ (root)` folder.
5. Save. GitHub Pages will publish `index.html`.

## Local preview

You can open `index.html` directly in a browser, or serve the folder with a
simple static server.
